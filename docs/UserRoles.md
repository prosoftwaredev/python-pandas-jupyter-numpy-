# User Roles

## Agency user
- can view their own dashboard
- can drill down in to their data and view associated graphs and charts
- can view comparative, deidentified data
- can export their data

## Finance user
- can view all agency dashboards
- can perform ad-hoc analysis
- can export all agency data
- can view identified, comparative analysis

## System administrator
- can upload new CSV data
- can publish a new release
